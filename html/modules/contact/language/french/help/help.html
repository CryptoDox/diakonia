Aide : Contact

Description


« Contactez-nous » est un module très simple. Il offre un lien à partir du menu principal vers un formulaire de contact que les visiteurs peuvent utiliser pour envoyer un courriel à l'administrateur du site. Vous pouvez définir le contenu du formulaire de contact dans « Préférences »

Installer / désinstaller


Aucune mesure spéciale nécessaire, suivez le processus d'installation standard – extraire le dossier /contact vers le dossier ../modules. Installer le module à partir d'Administration -> Modules Système -> Modules.

Des instructions détaillées sur l'installation de modules sont disponibles dans le Manuel des Opérations de XOOPS

Mode d'emploi


Il n'y a rien vraiment à faire sur le site d'administration, à part définir les préférences.
